  2. Implementation primes over specification.

  4. Function composition is modularity.
  5. Good abstractions don't lie.
  6. Compilers can't be fully trusted.
  7. The best API is a memory buffer.
  8. Performant code is code that doesn't exist.
  9. To protect a data, do not keep it.
